HONN + Adaptive EKF

A unique model designed and produced by Dr. Mishra — a Higher Order Neural Network trained via an Extended Kalman Filter based adaptive learning algorithm for nonlinear prediction. Full derivation, model development, and simulation, benchmarked against back propagation and standard MLP baselines.
